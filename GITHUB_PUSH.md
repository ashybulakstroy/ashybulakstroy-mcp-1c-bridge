# Publish to GitHub

Create an empty repo:

https://github.com/new

Repository name:

```text
ashybulakstroy-mcp-1c-bridge
```

Then run:

```bash
git init
git add .
git commit -m "Initial release: AshybulakStroy MCP 1C Bridge"
git branch -M main
git remote add origin https://github.com/AshybulakStroy/ashybulakstroy-mcp-1c-bridge.git
git push -u origin main
```
